Algunas variables se actualizan dinámicamente, por ejemplo LINES y

4.4. Exportar variables
Las variables de entorno que definimos dentro de una sesión de Bash no
están disponibles para los subprocesos que lanza Bash (sólo las puede usar
Bash y nuestros comandos interactivos y scripts) a no ser que las exportemos
con el comando export. Por ejemplo:
$ EDITOR=/sw/bin/joe
$ export EDITOR
También podemos exportar una variable de entorno a la vez que la
asignamos un valor así:
$ export EDITOR=/sw/bin/joe
Podemos obtener un listado de las variables exportadas usando el comando
env o el comando export sin argumentos. Para obtener las variables tanto
exportadas como no exportadas debemos usar el comando set sin
argumentos.
